OPTION B — cron-job.org (ACTIVE SETUP)
=====================================

One-time setup (5 minutes):

1. cron-job.org account (free)
   https://cron-job.org/en/signup/

2. API key
   https://console.cron-job.org/settings → Create API key

3. GitHub Personal Access Token (classic)
   https://github.com/settings/tokens/new
   Scope: repo (full control of private repositories)

4. Add to .env in this folder:

   TELEGRAM_TOKEN=...
   TELEGRAM_CHAT_ID=...
   TELEGRAM_GROUP_CHAT_ID=...
   CRONJOB_API_KEY=...
   GITHUB_PAT=ghp_...
   DHAN_ACCESS_TOKEN=...          # optional — live Nifty option premium
   DHAN_CLIENT_ID=...
   UPSTOX_ACCESS_TOKEN=...        # Apps → Analytics → Generate Token (required)
   UPSTOX_API_KEY=...
   UPSTOX_API_SECRET=...
   UPSTOX_WS_ENABLED=true         # WebSocket live LTP during session
   UPSTOX_AUTO_TRADE_ENABLED=true   # options only; use Telegram /live /paper /stop
   UPSTOX_PAPER_TRADE=false         # default LIVE when upstox_live_runner.py starts
   TELEGRAM_COMMANDS_ENABLED=true

LIVE TRADING (PC / VPS — not GitHub Actions)
   python upstox_live_runner.py     # starts LIVE + WebSocket + scanner
   python upstox_live_runner.py --paper   # start safe, then send /live in Telegram

TELEGRAM COMMANDS (in bot chat)
   /live   — real Upstox option orders (entry + SL + target)
   /paper  — test mode (no broker orders)
   /stop   — disable Upstox orders
   /status — mode + Upstox connection
   /lots 1 — option lots (1-10)
   /help   — command list
   FYERS_APP_ID=...
   FYERS_SECRET_KEY=...
   FYERS_REDIRECT_URI=https://trade.fyers.in/api-login/redirect-uri/index.html
   # ^ must match My API app Redirect URL exactly (fixes redirectUrl mismatch)
5. Run setup (PowerShell):

   cd "g:\Code\Telegram Trading bot"
   python scripts/refresh_nse_universe.py    # NSE EQ Rs 100-1000 cache (once/day)
   python scripts/rebuild_watchlist.py       # F&O cache + filtered watchlist
   python scripts/setup_github_secrets.py    # push secrets to GitHub Actions
   python scripts/setup_cron_job_org.py      # create/update cron job
   python scripts/fyers_login.py             # browser login → FYERS_ACCESS_TOKEN
   python scripts/test_fyers.py               # optional — Fyers option quotes

6. Verify live scan

   python scripts/setup_cron_job_org.py --test

   → GitHub Actions run starts
   → Telegram signals / session messages as per schedule

SIGNAL LIFECYCLE (scanner)
--------------------------
  • data/active_positions.json carries open equity + premium plans and automatically checks
    delayed 5m cash prices / option LTP before queuing new ideas.
  • Long BUY: MAX_DAILY_BUY_SIGNALS total per IST day (5–10, default 10) plus MAX_BUY_ALERTS_PER_SCAN
    each 5m run (default 1), so buys are spread through the session, not blasted in one tick.
  • Short SELL: MAX_SELL_ALERTS_PER_SCAN per 5m run only (no daily cap). Use 0 to disable (GitHub default); set 1–5 locally if you want shorts.
  • Telegram includes a Flag line when a symbol reappears after TP/SL on an earlier plan.
  • Set SLTP_CLOSE_ALERT_TELEGRAM=true for extra TP/SL clear pings.
