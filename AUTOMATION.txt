OPTION B — cron-job.org (ACTIVE SETUP)
=====================================

One-time setup (5 minutes):

1. cron-job.org account (free)
   https://cron-job.org/en/signup/

2. API key
   https://console.cron-job.org/settings → Create API key

3. GitHub Personal Access Token (classic)
   https://github.com/settings/tokens/new
   Scope: repo (full control of private repositories)

4. Add to .env in this folder:

   CRONJOB_API_KEY=paste_cron_job_api_key
   GITHUB_PAT=ghp_paste_github_token

5. Run setup (PowerShell):

   cd "g:\Code\Telegram Trading bot"
   .\scripts\setup_cron_job_org.ps1

   This creates job: "Telegram Trading Bot — NSE every 5 min"
   Schedule: Mon-Fri, every 5 min, 9:00-15:55 IST

6. Verify

   python scripts/setup_cron_job_org.py --test

   → GitHub Actions run starts
   → Telegram: scan result or "No signal right now"

COMMANDS
--------
  python scripts/setup_cron_job_org.py --list   # list cron-job.org jobs
  python scripts/setup_cron_job_org.py --test   # one scan now

TELEGRAM
--------
  • BUY/SELL signals → instant (Chaitu50c, Nifty 500, 0.6% max SL, 1:2 R:R)
  • Pre-market report → ~9:10–9:26 IST
  • Session START / STOP → market open / after 3:30 PM IST
  • Bot RUNNING ping → ~30 min after session start (signals not delayed)

NOTE: cron-job.org free tier = 100 API calls/day (~78 scans/day fits).
